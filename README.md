## Reset Categories

### Daily Resets (00:00 UTC)
- Daily Achievements
- Login Rewards
- World Boss Chests (Tequatl, etc.)
- Fractals
- Strike Missions
- Home Instance Nodes
- Jumping Puzzles

### Weekly Resets (Monday 07:30 UTC)
- Raid Wings
- Guild Missions
- Strike Mission Weekly Chests
- PvP/WvW Weekly Chests
- Legendary Armory Limits

### WvW Resets
- Matchup Reset (Friday 18:00 UTC)
- Skirmish Reset (Every 2 hours)

### Meta Events
- Heart of Thorns Maps
- Path of Fire Maps
- Living World Maps
